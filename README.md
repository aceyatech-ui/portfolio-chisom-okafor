# Aceya — Portfolio

> Personal portfolio website for Chisom Okafor (Aceya / Slime) — AI/ML, Automation & Dev Engineer.

Live demo: [https://aceyatech-ui.github.io/portfolio-chisom-okafor/](https://aceyatech-ui.github.io/portfolio-chisom-okafor/)

---

## 🚀 Features

- **Portfolio** — About, Skills, Projects, Ventures, Work Experience, Programs & Certifications, Reviews & Testimonials, Contact
- **Light / Dark Mode** — Persisted via `localStorage`
- **Chatbot** — Floating assistant with greeting options, jokes (max 3), quick replies, and Yes/No fallback
- **Review System** — Visitors submit reviews → pending approval in admin panel
- **Admin Panel** — Hidden, password-protected dashboard to approve/reject reviews
- **Hobbies Section** — Horizontal scroll with images and captions
- **Venture Roles** — Each venture displays your role (e.g., Founder & CEO) as a capsule badge
- **Fully Responsive** — Works on mobile, tablet, and desktop
